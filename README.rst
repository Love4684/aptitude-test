`Reverse Words in a String <https://leetcode.com/problems/reverse-words-in-a-string/>`_
===============================================================================


1. If log 2 = 0.3010 then value of log 25 is
===============================================================================

.. code:: math
 
    log 25=log5 + log5
    =log (10/2)+log (10/2)
    =(log 10-log 2)+(log 10-log 2)
    =(1-0.3010)+(1-0.3010)
    =0.6990+0.6990
    =1.3980
